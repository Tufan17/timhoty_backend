Timhoty Api's

    login: /auth/admin/login - POST 
    logout: /auth/admin/login - POST 

    Admin
        datatable: /admins - GET                        -> Adminlerin çağrıldığı datatable
        findOnly: /admins/:id - GET                     -> Admin verileri
        create: /admins - POST                          -> Yeni admin oluşturma
        update: /admins/:id - PUT                       -> admin güncelleme
        delete: /admins/:id - DELETE                    -> admini silme

    Permission
        findUser: /permissions/:target/:id - GET        -> target: users || admins || solution_partner || sale_partner id: kulannıcı id => Kulanıcının izinlerini getirir
        createOrUpdate: /permissions - POST             -> kullanıcı izni yoksa ekle varsa güncelle
        totalCreateOrUpdate: /permissions/total - POST  -> { name: string; status: boolean }[] bu şekilde toplu işlemler yapar 
    
    Log
        findAll: /logs - GET                            -> logların datatable'ı 
        findOne: /logs/:id - GET                        -> bireysel log datası
    
    Language
